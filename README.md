1. Введение:
- Docker Compose (утилита для управления несколькими контейнерами).
- Конфигурация осуществляется через файл docker-compose.yml.

2. Команды запуска:
- Базовая команда: ```docker-compose up```. Данная команда собирает образы (если нужно) и запускает все сервисы, описанные в файле.
- Фоновый запуск: ```docker-compose up -d```. Данная команда для запуска контейнеров в фоновом режиме, чтобы терминал остался свободным.
